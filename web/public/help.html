Getting Started

UltraChartWeb is a financial charting application designed for analyzing security price data with unique astrological planet line overlays. The interface follows a familiar MDI (Multiple Document Interface) pattern with a menu bar, toolbar, chart workspace, and status bar.

When you first open the app, you'll see an empty workspace. Load a chart using one of these methods:

  • Press Ctrl+L to open the Import/Load dialog
  • Go to File > Import / Load...
  • Load built-in sample data via Help > Load Sample: ZS Daily or ZS 5-min

Loading Charts

Sample Data (No Connection Required)

Use Help > Load Sample: ZS Daily or Help > Load Sample: ZS 5-min to instantly load built-in Soybean (ZS) futures data. This works without any TWS connection.

From TWS (Interactive Brokers)

If the TWS bridge server is running and connected to Trader Workstation:

  1. Press Ctrl+L or use File > Import / Load...
  2. Search for a symbol (e.g., "ZS" for Soybeans)
  3. Select the contract from the search results
  4. Choose the time interval (5-min, 15-min, daily, etc.)
  5. Click Import to download historical data

From Cache Files

Previously downloaded data is saved as cache files. In the Import dialog, the Cache tab lists all available cache files. Click one to load it, or use Sync to update it with the latest bars before loading.

From .uchart Files

Use File > Open File... (Ctrl+O) to open a previously saved .uchart file, which preserves chart settings, drawing objects, and planet lines.

Color, Style & Font Controls

The right side of the toolbar contains style controls that affect newly drawn objects:

Color Palette

Click a color swatch to set the drawing color. The active color shows a highlighted border. Available colors include blues, reds, greens, yellows, purples, oranges, cyans, pinks, and grays/white/black.

Line Style

Choose between three line dash patterns:

  • Solid — continuous line
  • Dashed — long dashes
  • Dotted — small dots

Line Width

Choose from 1px, 2px, 3px, or 4px line thickness.

Font

The font dropdown on the far right sets the typeface for text labels. Options include Arial, Helvetica, Verdana, Times New Roman, Courier New, and more.

Planet Lines (Astro)

Planet lines are a unique feature that overlays astrological planet position curves onto your price chart. These are used in financial astrology to identify potential price correlations with planetary cycles.

Adding a Planet Line

  1. Click the planet line icon (curved line) in the toolbar, or use Insert > Planet Line...
  2. In the dialog, select a planet (Sun, Moon, Mercury, Venus, Mars, Jupiter, Saturn, etc.)
  3. Configure the calculation parameters (coordinate type, scaling)
  4. Click Add to overlay the planet line on the active chart

Ephemeris Providers

Planet positions are calculated using one of two backends:

  • Swiss Ephemeris (WASM) — high-precision astronomical calculations loaded as a WebAssembly module. This is the default when available.
  • Equations — simplified mathematical approximations used as a fallback if Swiss Ephemeris fails to load.

You can switch providers in Edit > Preferences. The current provider is shown in the status bar.

Time Modes & Session Bands

Time Modes

Charts support two time display modes, toggled with T or the compress button in the chart header:

  • Natural Time — bars are positioned proportionally by timestamp. Gaps appear between trading sessions (weekends, overnight). This gives a true time-proportional view.
  • Compressed Time — bars are packed sequentially by index with no gaps. This is the traditional charting view that maximizes data density.

Session Bands

In Natural Time mode, colored background bands highlight different trading periods:

  • Sleep band (reddish) — overnight period (00:00 – 05:45 local time)
  • Lunch band (bluish) — midday period (06:00 – 11:15 local time)

Toggle session bands via View > Show/Hide Session Bands.

Live Streaming

When connected to TWS via the bridge server, you can receive real-time price updates:

  1. Load a chart from a TWS-sourced cache file
  2. Click the LIVE button in the top-right of the menu bar
  3. The button turns green with a pulsing dot when streaming is active
  4. New bars will appear in real time as they arrive
  5. Click LIVE again to stop streaming

The status bar shows LIVE when streaming is active for the current chart.

Preferences

Open via Edit > Preferences...

Ephemeris Provider

Choose between Swiss Ephemeris (high-precision WASM) and Equations (mathematical fallback). The WASM status badge shows whether the Swiss Ephemeris module loaded successfully.

TWS Connection

Shows the configured TWS bridge server host and port (read-only). The bridge server must be started separately.
